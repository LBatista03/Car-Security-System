# Car Security System

This project implements a real-time embedded system based on Linux using a Raspberry Pi to monitor a car for theft attempts. If an intrusion is detected, an SMS notification is sent to the user, specifying which sensors were triggered and the vehicle’s current status. In the event of a successful theft, the system enables real-time GPS tracking, providing continuous updates on the vehicle’s location and sending its coordinates to the user. Additionally, all relevant data, including alerts and location updates, is stored in a database for further analysis.
